RESOLUSI BELAJAR
- GITHUB
  (COMMIT) : merekam kejadian pada repo
  * menambahkan file di github
  * edit dan hapus file di github
  (BRANCH) : membuat cabang dari repo di jalur master (awal)
  (MERGE) : menggabungkan cabang (branch) dari repo di jalur master (awal)
  (HASH) : kode unik yg terdapat setiap (COMMIT)
  (CHECKOUT) : kembali ke commit yang kita pilih dengan (HASH)
  * membuat branch di github
  * menggabungkan branch yang kita buat ke dalam master
  (FORK) : menduplikat atau copy repo github lain ke dalam repo kita
